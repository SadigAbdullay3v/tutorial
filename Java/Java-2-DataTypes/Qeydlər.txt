Bəzi data type lara baxaq:

int -> tam natural rəqəmləri özündə saxlayan data type dır (bölmə əməlyatında onluq kəsir şəkilində qalığı göstərmir).
double -> tam və onluq kəsrləri hesablama əsasəndə bölmə imkanı verir.
char -> yanlız tək bir simvol print etməyimizə kömək edir.
string -> hərfdən daha çox yəni cümlə və ya söz yazmağımıza imkan verir.
boolean -> özündə iki dəyər saxlayır true və false.

qeyd : qoşa dırnaq ("") içində yazdığımız rəqəmlər simvollar belə string type'na aiddir.
qeyd : tək dırnaq ('') əsasən char data type'n da tək bir simvol print etmək üçün isdifadə olunur.
qeyd : sout + tab yazaraq  System.out.println(); yazmaq mümkündür.Əgər sout yazıb tab düyməsinə vursas və heçbir nəticə
almasaz sout yazdıqdan sonra boşluq buraxmadan ctrl+space düyməsinə ard-arda klikləyin və enter düyməsinə basın
